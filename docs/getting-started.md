# Getting Started

This guide helps teams adopt the workflow templates in under one hour.

## 1) Choose a starting template
Pick one template from `templates/` based on your immediate business need:
- Customer Support
- Content Ops
- Lead Enrichment
- Bug Triage
- Incident Response

## 2) Define connectors
Map each step in the template to your tools (CRM, Helpdesk, CMS, Issue Tracker, Monitoring).

## 3) Run a pilot
Start with a small scope (one team, one queue, one product area) for 1–2 weeks.

## 4) Measure outcomes
Track template KPIs weekly and compare against baseline metrics.

## 5) Iterate
Review bottlenecks, update step definitions, and publish versioned improvements.
