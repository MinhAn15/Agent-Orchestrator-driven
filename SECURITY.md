# Security Policy

## Reporting a vulnerability
Please report vulnerabilities privately to maintainers. Include:
- Affected files/components
- Reproduction steps
- Potential impact
- Suggested remediation (if known)

## Response targets
- Initial acknowledgement: within 72 hours
- Triage and risk assessment: within 7 days
- Mitigation plan communication: as soon as available

## Disclosure
Please avoid public disclosure until a fix or mitigation is available.
