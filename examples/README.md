# Examples Matrix

Use this matrix to quickly map a business use-case to a workflow template and suggested connectors.

| Use-case | Template | Suggested connectors |
|---|---|---|
| Handle support tickets with SLA | `templates/customer-support.md` | Zendesk/Freshdesk, CRM, Knowledge Base, Slack |
| Produce SEO blog posts weekly | `templates/content-ops.md` | Notion/Docs, SEO tool, CMS, Analytics |
| Prepare outbound lead list | `templates/lead-enrichment.md` | Enrichment API, CRM, Email validation, BI dashboard |
| Prioritize engineering defects | `templates/bug-triage.md` | Jira/Linear, Logs/APM, Error tracking, Slack |
| Manage production outage | `templates/incident-response.md` | PagerDuty/Opsgenie, Monitoring, Status page, ChatOps |
