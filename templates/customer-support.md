# Customer Support Workflow Template

## Goal
Handle incoming customer requests consistently, reduce response time, and ensure resolution quality.

## Inputs
- Ticket source (email, chat, CRM)
- Customer profile and plan tier
- Product knowledge base and troubleshooting guides

## Workflow (Copy/Paste)
1. Classify request type (billing, technical, account, feature request).
2. Auto-retrieve account context and recent interactions.
3. Suggest top 3 solutions from knowledge base.
4. Draft reply with empathetic language and clear next steps.
5. If unresolved in 2 replies, escalate to specialist queue.
6. Close ticket with resolution tag and customer sentiment.

## Output
- Customer response draft
- Internal resolution note
- Ticket status update and tags

## KPIs
- First response time
- Time to resolution
- CSAT score
